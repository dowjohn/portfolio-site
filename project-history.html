Project History

Rigado Toolbox


While at Rigado I contributed to the codebase of the Rigado Toolbox company app. Rigado Toolbox provides a way for Rigado customers to configure their Bluetooth modules as iBeacons and other out of the box firmware features.

BrewCheck


An Android application that leverages the BreweryDB API providing a way to search an area for breweries using either a input or the user's current location. The user may also submit reviews for breweries and view other reviews for breweries.

PewPewPew


This is the culmination of the Android class at Epicodus. Our group chose to do something extraordinary and created a Unity2d game from scratch with no prior experience using Unity2d. I wrote the C# scripts as well as helped design the game.
